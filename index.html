Features

PARITRATA Flood Relief.

Flood Early Waring System

Flood alert system which would enable the detect the flash flood that is occurring in an area, using an IoT framework

Flood Effected Zone on Google Map

Flood effected area will be shown directly on google map based on Flood intensity in both APP and DashBobard

Live Water drainage Monitoring

Live water drainge data will available on Govt. officials DashBorad and it will update in Google maps as well in frequent times

Mobile App for Real Time tracking

Mobile application will help for tracking flood on nerby places as well as many other visual and statical information

Live Alternative Route To Prevent Flooded Area

Alternative route suggesting between destination avoiding the flood routes and also analysing the time for flash floods to clear..

Prediction Model Using Machine Learing

Using past archive data, with sensor data and weather forecast, machine learning algorithm is applied to predict flood situations.

Live Streming of Flood Effected Area

With cameras at different zones, flood zones can be monitored from the dashboard itself, available only for agencies.

DashBoard for Govt. Official

Web dashboard with all the necessay data with data archives for monitoring and study on the flood situations.

Emergency Services

Emergency services during flood can be requested using either SMS gateways or Mobile applications, with panic wave support.

Future Scopes

PARITRATA Flood Relief.

UAV Drone Algorithm

GPS guided drone with heat sensing camera for identifying trapped flood victims over a large area and notifying relief teams

Low Level Mobile Broadcast

Network broadcasts to peoples living near dam reservoir irrespective of their carrier and mobile phone they are using.

LoRa based communication

LoRa based long range communication system between relief teams, independent of mobile networks or satellite uplinks.
